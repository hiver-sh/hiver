# Agent CLI

Hiver ships the popular agent CLIs, each in a ready-to-run image. Pick one, provision its image, pass the right credential as an env var, and drive it with `exec` (one-shot) or `execStream` (live output).

<Tabs>
<Tab label="Claude Code">

Run [Claude Code](https://claude.com/claude-code) inside a sandbox. The `claude` image ships the CLI ready to go.

### One-shot task

`claude -p` runs a single prompt non-interactively and prints the result. Use `exec` when you just need the final output.

<CodeBlock tabs={[
  { label: "TS", lang: "typescript", code: `import { getOrCreateSandbox } from "@hiver.sh/client";

const sandbox = await getOrCreateSandbox("claude", {
  image: "claude",
  env: { ANTHROPIC_API_KEY: process.env.ANTHROPIC_API_KEY! },
});

const result = await sandbox.exec(["claude", "-p", "Fix the bug in src/main.ts"], {
  cwd: "/workspace",
});
console.log(result.stdout);` },
  { label: "PY", lang: "python", code: `import os
from hiver import get_or_create_sandbox, SandboxConfig

sandbox = await get_or_create_sandbox("claude", SandboxConfig(
    image="claude",
    env={"ANTHROPIC_API_KEY": os.environ["ANTHROPIC_API_KEY"]},
))

result = await sandbox.exec(["claude", "-p", "Fix the bug in src/main.ts"], cwd="/workspace")
print(result["stdout"])` },
  { label: "Go", lang: "go", code: `c := client.NewClient(client.DefaultGatewayURL)
sandbox, _ := c.GetOrCreateSandbox(ctx, "claude", client.SandboxConfig{
    Image: "claude",
    Env:   map[string]string{"ANTHROPIC_API_KEY": os.Getenv("ANTHROPIC_API_KEY")},
})

result, _ := sandbox.Exec(ctx, client.ExecRequest{
    Command: []string{"claude", "-p", "Fix the bug in src/main.ts"},
    CWD:     "/workspace",
})
fmt.Println(result.Stdout)` },
  { label: "Command line", lang: "bash", icon: "bash", filename: "Terminal", code: `hiver start claude <<EOF
{
  "image": "claude",
  "env": { "ANTHROPIC_API_KEY": "$ANTHROPIC_API_KEY" }
}
EOF
hiver shell claude --command "claude -p 'Fix the bug in src/main.ts'"` },
]} />

The key is injected once at creation and is immutable, every later `exec` on this sandbox inherits it automatically, so you never pass it again.

### Streaming output

Agents can run for a while and produce a lot of output. Use `execStream` to see stdout and stderr as they happen instead of waiting for the process to finish.

```typescript
const proc = await sandbox.execStream(["claude", "-p", "Refactor the auth module"], {
  cwd: "/workspace",
});

for await (const chunk of proc.pipes) {
  if (chunk.stdout) process.stdout.write(chunk.stdout);
  if (chunk.stderr) process.stderr.write(chunk.stderr);
}

console.log("exit code:", await proc.exitCode);
```

```python
proc = await sandbox.exec_stream(["claude", "-p", "Refactor the auth module"], cwd="/workspace")

async for chunk in proc.pipes:
    if "stdout" in chunk:
        sys.stdout.write(chunk["stdout"])
    if "stderr" in chunk:
        sys.stderr.write(chunk["stderr"])

print("exit code:", await proc.exit_code)
```

```go
proc, _ := sandbox.ExecStream(ctx, client.ExecStreamRequest{
    Command: []string{"claude", "-p", "Refactor the auth module"},
    CWD:     "/workspace",
})
for chunk := range proc.Output {
    fmt.Print(chunk.Stdout)
    fmt.Fprint(os.Stderr, chunk.Stderr)
}
code, _ := proc.Wait()
fmt.Println("exit code:", code)
```

### Interactive TUI

To drive Claude Code's full-screen interface from your own terminal, attach over a PTY, see [Pseudo Terminals](/docs/pseudo-terminals) for a complete example.

<Callout>Keep secrets out of the agent's reach: rather than giving Claude your third-party API keys, allow the hosts it needs and inject credentials with an egress [override](/docs/network/overrides), the agent uses them without ever seeing them.</Callout>

### Streaming output

For longer runs, stream output as it's produced with `execStream`:

```typescript
const proc = await sandbox.execStream(["codex", "exec", "Migrate the app to ESM"], {
  cwd: "/workspace",
});
for await (const chunk of proc.pipes) {
  if (chunk.stdout) process.stdout.write(chunk.stdout);
}
console.log("exit code:", await proc.exitCode);
```

```python
proc = await sandbox.exec_stream(["codex", "exec", "Migrate the app to ESM"], cwd="/workspace")
async for chunk in proc.pipes:
    if "stdout" in chunk:
        sys.stdout.write(chunk["stdout"])
print("exit code:", await proc.exit_code)
```

```go
proc, _ := sandbox.ExecStream(ctx, client.ExecStreamRequest{
    Command: []string{"codex", "exec", "Migrate the app to ESM"},
    CWD:     "/workspace",
})
for chunk := range proc.Output {
    fmt.Print(chunk.Stdout)
}
code, _ := proc.Wait()

```

The `OPENAI_API_KEY` you set at creation is inherited by every later `exec`, so you set it once. To drive Codex's interactive UI from your terminal, attach over a PTY, see [Pseudo Terminals](/docs/pseudo-terminals).
