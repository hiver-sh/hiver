# Installation

## Requirements

- Node.js 18+
- Docker

## Install the CLI

<CodeBlock tabs={[
  { label: "npm", lang: "bash", icon: "bash", filename: "Run", code: "npm i -g @hiver.sh/cli" },
  { label: "Bash", lang: "sh", icon: "bash", filename: "Run", code: "curl -fsSL https://hiver.sh/install | sh" },
  { label: "PowerShell", lang: "shell", icon: "bash", filename: "Run", code: "irm https://hiver.sh/install.ps1 | iex" },
]} />

## Install the client SDK

Add the Hiver client to your project to provision and control sandboxes from code.

<CodeBlock tabs={[
  { label: "TS", lang: "typescript", icon: "bash", filename: "Run", code: "npm install --save @hiver.sh/client" },
  { label: "PY", lang: "python", icon: "bash", filename: "Run", code: "pip install hiver-py" },
  { label: "Go", lang: "go", icon: "bash", filename: "Run", code: "go get github.com/hiver-sh/hiver/client" },
]} />

## Agent skill

The Hiver [Agent Skill](/docs/cli#install-the-skill) — the full documentation as an on-disk skill — ships inside the CLI. Wire it into the coding agents on your machine so they can pull these docs into context on demand:

<CodeBlock tabs={[
  { label: "Bash", lang: "sh", icon: "bash", filename: "Run", code: "hiver install-skill" },
]} />

It detects Claude Code, Codex, Antigravity, and GitHub Copilot CLI. The first run of `hiver` also offers to install it.
